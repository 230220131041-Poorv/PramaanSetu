## No Setup Required

1. Just run the app normally
2. Click on any role card on the login screen
3. You're logged in instantly
4. No database, no API keys, no internet needed

## Session Management

- Sessions are stored in browser's localStorage
- Sessions persist across page refreshes
- Sessions are cleared when cache is cleared
- Each role has a separate profile

## Notes

- This is a demo/development mode
- All data is stored locally
- No external API calls are made
- Perfect for UI/UX testing and feature development
